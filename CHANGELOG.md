# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-22

### Added
- Initial release of DQN Atari implementation
- DQN training script with 10 hyperparameter experiments
- Policy comparison (MlpPolicy vs CnnPolicy)
- Gameplay inference module with GUI support
- Jupyter notebooks for interactive experimentation
- Comprehensive configuration system
- Professional project structure
- Unit test framework
- Contributing guidelines
- Full documentation

### Features
- Deep Q-Networks (DQN) for Atari environments
- Hyperparameter tuning utilities
- Model checkpointing and evaluation
- TensorBoard logging integration
- Extended experiments (exp11-exp20) with infrastructure variations
- Gameplay video recording and visualization
